# zTheme

Modern, smooth and elegant Visual Studio Code theme with a beautiful blue color palette.

![Theme Banner](https://github.com/cycscarlos/myTheme-VSC/raw/HEAD/img/theme-banner.jpg)

## Features

- **Smart palette colors** for both day and night coding sessions
- **Smooth blue background** for the editor
- **Elegant contrast gold color** for HTML, CSS classes and JavaScript items
- **Pleasant palette colors** for programming languages like JavaScript
- **Clean and modern design** with carefully selected color combinations

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "zTheme"
4. Click Install
5. Go to File > Preferences > Color Theme (or Code > Preferences > Color Theme on Mac)
6. Select "zTheme"

## Color Palette

- **Editor Background**: Smooth blue (#1c3549)
- **Sidebar**: Dark blue (#0b273c)
- **Accent Colors**: Gold (#9d8d6c) for HTML/CSS classes
- **Syntax Highlighting**: Carefully tuned colors for optimal readability
